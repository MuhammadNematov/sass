# Social Network

Современный дизайн социальной сети, созданный в Figma.  
Проект включает стильный интерфейс, адаптивный дизайн и удобную структуру страниц для общения пользователей.

## О проекте

Этот проект представляет собой дизайн социальной платформы с современным UI/UX.

В макете присутствуют:
- главная страница
- профиль пользователя
- чат / сообщения
- лента публикаций
- блок друзей
- навигация
- адаптивная мобильная версия

Дизайн выполнен в минималистичном современном стиле с акцентом на удобство и читаемость интерфейса. :contentReference[oaicite:0]{index=0}

## Технологии

- CSS3

## Особенности

- Адаптивный дизайн
